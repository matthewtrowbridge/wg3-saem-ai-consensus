# Source materials

Durable copies of the source documents that the WG3 working site draws on. Lives inside the repo so that a clone, fork, or org migration carries the supporting material with it — not just the rendered HTML.

## What's here

- **`AI_Competency_Frameworks_Brief.md`** — Matt Trowbridge's April 6, 2026 synthesis of published AI competency frameworks for medical education (Triola & Rodman 2025, Lee et al. 2024, AAMC CGEA, Tolentino et al. 2024, AI-PACE 2026, AIFM-ed 2025, AMEE Guide 178). Provides the landscape context for WG3's Aim 3 work and frames the "what's missing — WG3's unique contribution" positioning for the AEM E&T manuscript. Referenced from `resources.html` Literature landscape card.

## Copy policy

See the repo root `CLAUDE.md` for the rules on what belongs in this folder, what requires author permission, and what should only ever be linked (not hosted). In short:

- **Safe to copy:** Matt's own writing, public links, anything Matt authored from scratch.
- **Ask the author first:** Andrew's grant materials, Michael's prior work, anything derived from someone else's draft or pre-publication content.
- **Link, don't host:** Third-party copyrighted papers.
